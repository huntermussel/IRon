# Device Middleware

This middleware handles device control requests using the shared NLU engine.

## Intents

- **control_device**

## Utterances

- "turn {state} the {device}"
- "turn {state} {device}"
- "switch {state} the {device}"
- "switch {state} {device}"
- "turn the {device} {state}"
- "turn {device} {state}"

## Example

User: "turn on the lights"
Response: "Turned on the lights."
