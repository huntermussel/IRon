# Alarm Middleware

This middleware handles alarm setting requests using the shared NLU engine.

## Intents

- **set_alarm**

## Utterances

- "set alarm for {time}"
- "set an alarm for {time}"
- "wake me up at {time}"
- "create alarm for {time}"
- "alarm {time}"

## Example

User: "wake me up at 7am"
Response: "Alarm set for 7am."
